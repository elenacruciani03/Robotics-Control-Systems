# Robotics Engineering Portfolio

Welcome! This repository showcases my work in robotics, focusing on control theory and state estimation.

## Projects

### [TurtleBot3 Control Law](./turtlebot3-control)
A Python implementation of a trajectory tracking controller for a mobile robot in ROS1/Gazebo.

### [Indoor Positioning & Kalman Filter](./indoor-positioning-kalman)
A full-stack project (hardware + software) using Arduino and Python to create a real-time positioning system smoothed by a Kalman Filter.
